Slide

O slide, já está configurado com as classes e id's necessários para o seu funcionamento. Caso não deseje utilizar controles, basta retirar o elemento que possui a classe custom-controls e acrescentar uma classe no-controls, no elemento que possui a classe slide-wrapper. Para retirar as setas, deve-se retirar o elemento com a classe arrow-nav

Para adicionar mais de um slide na mesma página, deverá ser utilizado o node js configurado com o gulp, explicado nessa documentação.

HTML Exemplo

   <ul class="custom-controls" id="custom-controls">
