Documentação

Arquivos

Para utilizar o projeto, faça o download dos arquivos.

Insira o arquivo components.css no header do html e default.css, caso deseje utilizar os estilos padrões. Adicionar também o arquivo components.js antes de fechar a tag body no html.

NPM

É possível utilizar o web-components via NPM. Para isso, acesse o repositório do github web-components, clone o projeto e faça um npm install.

Ele já vem estruturado com pug, sass e babel no gulpfile. O código é aberto e pode ser totaltmente modificado. Para rodar o projeto, digite gulp na linha de comando.

Mais slides na página

Para adicionar mais de 10 slides na mesma página, clone o repositório no git, faça um npm install e acesse o arquivo main.js. Acrescente novas constantes nos construtores dos slides e modifique os valores de data.

Obs: No HTML os elementos devem ter o mesmo nome das constantes do main.js.
