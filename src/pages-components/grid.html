Grid

Veja como utilizar

Para utilizar o sistema de grid é necessário adicionar as classes grid-numero*, grid-xs-numero*, grid-s-numero*, grid-m-numero* e grid-l-numero*.

O número varia de 1 até 12, pois foi desenvolvido para comportar até 12 colunas.

  • O valor do grid- corresponde à tela com tamanho até 560px;
  • O valor do grid-xs (extra-small) corresponde à tela com tamanho mínimo de 560px;
  • O valor do grid-s (small) corresponde à tela com tamanho mínimo de 768px;
  • O valor do grid-m (medium) corresponde à tela com tamanho mínimo de 992px;
  • O valor do grid-l (large) corresponde à tela com tamanho mínimo de 1200px;

Para acrescentar uma margem de offset, é necessário adicionar as classes grid-offset-numero*, grid-xs-offset-numero*, grid-s-offset-numero*, grid-m-offset-numero* e grid-l-offset-numero*.

O sistema foi desenvolvido com uso do CSS Grid Layout e portanto, não é compatível com todos os browsers, principalmente os mais antigos.

No site do Can I use é possível verificar à compatibilidade.
